# Rela-she-on

<h3>Project ditujukan untuk keperluan tugas HACTIV8 - AI Productivity and AI API
Integration for Developers</h3>

ini adalah simple project terkait penggunaan ai api google. Project ini dibuat denga teknologi sederhana dan berfokus pada teknik iplementasi apinya daripada elegansi aplikasi (wlaau kedepanya bisa dibuat lebih matang). Untuk menjalankan aplikasi ini sederhana. Strukturnya 
folder: <br/> 
<ul>
  <li>folder STARTER adalah FE nya</li>
  <li>folder SERVER adalah BE nya</li>
</ul>
<br/>
Untuk menjalankannya:<br/>
<ol>
  <li>Masuk ke folder SERVER lalu copy-paste .env.example ke .env dan isi gemin api anda sendiri. Setelah itu klik <code>npm install</code> lalu <code>node index.js</code></li>
  <li>Masuk ke folder STARTER lalu copy-paste jalankan <code>live server</code></li>
  <li>Seharusnya <code>project terbuka otomatis so browser</code></li>
</ol>
